# 足球应用开发

Mini Pi+ 足球应用框架提供了完整的机器人足球开发解决方案，包含运动控制、视觉识别和策略规划三大核心模块。

## 架构概览

```
┌─────────────────────────────────────┐
│         足球应用层                    │
│  (策略决策、战术配合、角色分配)        │
└─────────────────────────────────────┘
           ↓           ↓
┌──────────────┐  ┌──────────────┐
│  视觉模块     │  │  运动模块     │
│ (目标识别)    │  │ (动作控制)    │
└──────────────┘  └──────────────┘
           ↓           ↓
┌─────────────────────────────────────┐
│         Mini Pi+ SDK                │
│    (传感器接口、执行器控制)           │
└─────────────────────────────────────┘
```

## 核心功能

### 🎯 视觉识别

实时识别和跟踪场上的关键目标：

- **足球检测**：基于颜色和形状的快速球体识别
- **球门定位**：识别己方和对方球门位置
- **队友识别**：区分队友和对手机器人
- **场地线检测**：识别边界线和中线

[查看视觉模块文档 →](/soccer/vision)

### 🤖 运动控制

流畅的运动控制算法：

- **行走控制**：全向行走，支持前进、后退、侧移和旋转
- **踢球动作**：多种踢球方式（正脚背、内脚背、挑射）
- **守门动作**：扑救、站位、出击
- **平衡控制**：动态平衡算法，防止摔倒

[查看运动控制文档 →](/soccer/motion)

### 🧠 策略规划

智能决策系统：

- **角色分配**：前锋、中场、后卫、守门员
- **行为树**：基于行为树的决策框架
- **路径规划**：避障和最优路径计算
- **团队协作**：多机器人协同策略

[查看策略规划文档 →](/soccer/strategy)

## 快速示例

以下是一个简单的足球应用示例：

```python
from minipi_soccer import SoccerRobot, Vision, Motion

# 初始化机器人
robot = SoccerRobot()

# 启动视觉系统
vision = Vision()
vision.start()

# 主循环
while True:
    # 获取视觉信息
    ball = vision.detect_ball()
    goal = vision.detect_goal()

    # 决策
    if ball.distance < 0.3:
        # 球在附近，踢球
        robot.motion.kick()
    else:
        # 走向球
        robot.motion.walk_to(ball.position)

    robot.update()
```
